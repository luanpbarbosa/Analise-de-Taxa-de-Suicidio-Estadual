# Análise-de-Taxa-de-Suicídio-Estadual
Trabalho realizado para a disciplina de Análise de Dados na Universidade do Estado de Minas Gerais, com o objetivo de praticar técnicas de análise de dados e aprendizagem de máquina.

A análise visa investigar a relação entre renda da população brasileira e taxa de suicídio, utilizando dados agregados a nível estadual no período de 2014 a 2023.

A análise exploratória realizada no conjunto de dados indicou que estados com maior nível de renda tendem a ter uma maior taxa de suicídio. Outros estudos realizados por pesquisadores a nível municipal encontrou resultados contrários. Entretanto, neste estudo foi possível perceber o impacto positivo na redução da taxa de suicídio da quantidade de unidades CAPS disponíveis para a população.

Também foi aplicado o modelo Random Forest no conjunto de dados a fim de identificar as variáveis mais impactantes no modelo, com destaque para o Índice de Vúlnerabilidade Social (IVS).

O grande desafio do projeto é a disponibilidade de dados, foi comum encontrar lacunas nos dados para algumas regiões brasileiras.
Um estudo mais aprofundado e com diferente nível de agregação pode gerar resultados diferentes dos que foram encontrados neste trabalho.
